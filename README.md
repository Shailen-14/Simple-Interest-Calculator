# 💰 Simple Interest Calculator

A simple and interactive **Simple Interest Calculator** built with **HTML, CSS, and JavaScript**.  
This project demonstrates front-end development skills, including **form handling**, **DOM manipulation**, and **basic UI design**.

---

## 🚀 Features
- Calculate **simple interest** instantly  
- User-friendly input fields for **Principal**, **Interest (%)**, and **Years**  
- Instant result display  
- Clean and responsive user interface  
- Lightweight and fast  

---

## 🛠️ Tech Stack
- **HTML** – Structure of the calculator  
- **CSS** – Styling and responsive design  
- **JavaScript** – Interactive logic and calculations  

---

## 📂 Project Structure
```
/InterestCalculator
├── index.html # Main HTML file
├── style.css # Styles for layout and design
├── script.js # JavaScript logic for calculation
```

---

## ▶️ How to Run
1. **Clone the repository:**
```bash
git clone https://github.com/Shailen-14/Simple-Interest-Calculator.git
```
2. Navigate to the project folder
3. Open index.html in your preferred web browser
4. Enter the Principal, Interest (%), and Years, then click Submit to see the calculated simple interest

---

## 🧠 How It Works

1. The user enters:
   - **Principal amount**
   - **Interest rate (%)**
   - **Time in years**

2. JavaScript calculates the **simple interest** using the formula:

```text
Simple Interest = (Principal × Interest × Years) / 100
```

3. The result is displayed instantly below the Submit button.
